💻 系统要求

项目要求
操作系统Windows 10/11 64位
手机系统Android 5.0+（音频转发需 Android 11+）
USB 调试需要开启
权限无需 root，无需在手机安装任何应用

开启开发者选项

  1. 手机 → 设置 → 关于手机
  2. 连续点击 版本号 7 次 → 提示「已开启开发者模式」
  3. 返回设置 → 系统 → 开发者选项
  4. 打开 USB 调试

部分手机（如小米）还需在开发者选项中开启 「USB 调试（安全设置）」 才能使用键鼠控制。

🚀 快速上手

安装

  1. 从 官网 点击下载安装包
  2. 双击安装，选择安装目录
  3. 安装完成后双击桌面快捷方式启动

首次使用流程

  1. 启动应用
  2. USB 连接手机
  3. 设备自动识别 → 点击设备选中
  4. 点「投屏」→「开始投屏」

🔌 USB 有线连接

推荐 最简单、延迟最低的方式。

  1. 用数据线将手机连接至电脑
  2. 手机上允许 USB 调试
  3. 打开 scrcpy GUI → 左侧自动显示设备
  4. 点击设备 → 侧边栏「投屏」→ 开始投屏

首次连接时，手机会弹出「允许 USB 调试吗？」对话框，勾选「始终允许」后确定。

📶 WiFi 无线连接

方法一：ADB 无线配对（推荐）

  1. 手机 → 开发者选项 → 无线调试 → 开启
  2. 点 「使用配对码配对设备」 → 记录 IP 端口和配对码
  3. 电脑运行：adb pair <IP>:<端口>
  4. 输入配对码 → 配对成功
  5. 在 scrcpy GUI 输入无线调试页面显示的 服务端口（非配对端口）
  6. 点击「连接」→ 开始投屏

方法二：USB 转无线

  1. 先用 USB 连接手机
  2. 运行：adb tcpip 5555
  3. 拔掉 USB 线
  4. 在 scrcpy GUI 首页输入手机 IP 地址
  5. 点「连接」→ 开始投屏

获取手机 IP 地址

  • 手机 → 设置 → 关于手机 → 状态信息 → IP 地址
  • USB 连接后运行：adb shell ip addr show wlan0

无线连接延迟比 USB 高，适合日常使用，游戏建议用 USB。

自定义预设

  1. 在「设置」页面调整好参数
  2. 点侧边栏「预设」
  3. 点「保存当前配置」→ 输入名称
  4. 下次直接点击预设即可一键应用

❓ 常见问题

Q: 提示「找不到设备」

确认 USB 调试已开启。换一根数据线（部分线只支持充电）。运行 adb devices 检查是否能识别。

Q: 投屏画面卡顿

使用 USB 有线连接 延迟最低。预设选择「低延迟模式」。降低分辨率和帧率。

Q: 没有声音

需要 Android 11+ 设备。检查设置中音频是否开启。部分小米机型音频转发可能有兼容问题。

Q: WiFi 连接失败

确认手机和电脑在同一个网络。确认手机已开启无线调试。尝试用 USB 先连接一次运行 adb tcpip 5555。

Q: 关闭按钮点了没反应

点击 × → 弹出「缩小到托盘」/「关闭应用」对话框。点「缩小到托盘」→ 窗口隐藏到系统托盘。左键点击托盘图标恢复窗口。右键托盘图标 → 退出。

Q: 应用启动报错

确保安装了 Microsoft Visual C++ Redistributable。如果报 electron-store 错误，请重新下载最新版本。

Q: 如何投屏不同网络下的手机？

使用 ZeroTier 或 Tailscale 等虚拟组网工具，将电脑和手机加入同一个虚拟网络，然后在 GUI 中输入虚拟 IP 连接。
