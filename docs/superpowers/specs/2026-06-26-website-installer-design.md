# scrcpy GUI 产品官网 + 安装包设计文档

日期: 2026-06-26

## 概述

为 scrcpy GUI 应用打造完整的分发体系：产品官网（静态页面）让用户了解并下载软件，NSIS 安装包让下载者能像安装普通软件一样完成安装配置。

---

## 一、产品官网

### 页面结构

参考 QQ 官网风格（im.qq.com），深色主题 + 蓝色主调。

| 模块 | 内容 |
|------|------|
| **导航栏** | Logo `📱 scrcpy GUI` + 功能特性 / 效果展示 / 下载安装 / 使用帮助 + 免费下载按钮 |
| **Hero 区域** | 渐变标题「scrcpy GUI」+ 副标题 + 醒目的渐变下载按钮 + 版本信息 |
| **特色卡片（4个）** | 无线投屏 / 音频转发 / 屏幕录制 / 极低延迟 |
| **截屏展示** | 应用界面截图 |
| **下载区** | Windows 64位下载按钮 + 系统要求说明 |
| **页脚** | 版权信息 + scrcpy 开源项目链接 |

### 技术实现

- 纯静态 HTML + CSS，无框架依赖
- 深色主题（#0f1419 背景），蓝色主色（#4A90D9）
- 响应式布局
- 下载按钮指向一个可下载的安装包文件

### 文件

- `website/index.html` — 官网首页
- `website/download/` — 安装包存放目录

## 二、安装包

### 需求

- 标准 Windows 安装程序（exe）
- 用户可选择安装路径
- 可选安装用户（所有用户/当前用户）
- 创建开始菜单和桌面快捷方式
- 安装进度条显示
- 安装完成后可选立即启动

### 技术方案

使用 **Inno Setup**（免费、轻量）或 **NSIS 脚本** 制作安装包，将 `dist/scrcpy-gui-portable/` 目录打包成安装程序。

### 打包内容

- Electron 运行环境（~180MB）
- scrcpy 主程序 + ADB 驱动
- scrcpy-server（Android 端）
- 应用图标

## 四、验收标准

1. 官网页面在浏览器中打开正常，导航跳转平滑
2. 下载按钮可点击（指向安装包）
3. 安装包双击运行后，显示安装向导界面
4. 可选择安装目录（默认为 Program Files\scrcpy GUI）
5. 可选择安装用户
6. 安装进度条正常显示
7. 安装完成后桌面和开始菜单出现快捷方式
8. 双击快捷方式能正常启动应用
